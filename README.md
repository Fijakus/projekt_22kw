# 🏠 Kalkulator Budżetu Domowego

Aplikacja do zarządzania budżetem domowym dla całej rodziny.

## Funkcje

- 👨‍👩‍👧‍👦 Zarządzanie członkami rodziny
- 💰 Śledzenie przychodów i wydatków
- 📊 Wykresy i statystyki
- 🎯 Cele oszczędnościowe
- 📅 Widok miesięczny
- 💾 Dane zapisywane lokalnie (localStorage)

## Uruchomienie

Otwórz plik `index.html` w przeglądarce.

## Technologie

- HTML5, CSS3, Vanilla JavaScript
- Chart.js (wykresy)
- Font Awesome (ikony)
- LocalStorage (przechowywanie danych)
